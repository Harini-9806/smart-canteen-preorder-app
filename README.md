# Smart Canteen Preorder Application

## Overview
A full stack web application to preorder food and reduce waiting time in canteens.

## Tech Stack
- HTML, CSS, JavaScript
- Node.js, Express.js
- MongoDB

## Features
- User authentication
- Menu browsing
- Cart system
- Order placement
- Order management

## How it Works
Users can browse menu, add items to cart, and place orders. Data is stored in MongoDB.

## Future Improvements
- Payment integration
- Admin dashboard
- Real-time tracking
